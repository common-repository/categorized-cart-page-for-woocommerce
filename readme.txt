== Description ==

This is a simple plugin design for woocommerce site owners who want to show product categories on cart page, with this plugin your cart page template will display categories of products and list of products inside the category.   

* Make your cart page more user friendly
* Just install and activate the plugin and you are ready to display a nice categorized cart page
* That's it.

= Features =

* Easy to use
* Display unlimited products 
* Display unlimited categories

== Installation ==

To install the plugin please follow these steps : 

1. Upload the plugin to the `/wp-content/plugins/` directory or use the Add New feature 
2. Activate the plugin through the "Plugins" menu in WordPress dashboard
